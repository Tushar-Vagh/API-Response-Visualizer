🚀 Project Overview

Response Prism is an interactive AI-powered text assistant built using modern frontend technologies. It allows users to generate creative, dynamic responses in real-time — blending advanced AI capabilities with an intuitive and responsive UI design.

Designed with simplicity and usability in mind, this project demonstrates how frontend logic can seamlessly integrate with AI services to deliver powerful user experiences.

▶️ How to Run the Project Locally

Follow the steps below to run Response Prism on your local machine.

1️⃣ Clone the Repository
git clone https://github.com/your-username/response-prism.git

cd response-prism

2️⃣ Install Dependencies

Make sure Node.js (v16 or above) is installed.

npm install


or if you use yarn:

yarn install

3️⃣ Start the Development Server
npm run dev


or:

yarn dev

4️⃣ Open in Browser

Once the server starts, open:

http://localhost:3000


You should now see the Response Prism application running locally 🎉

5️⃣ Build for Production (Optional)

To create a production build:

npm run build


To preview the production build:

npm run preview

📌 Notes

This project is frontend-only and does not require any backend setup.

All logic runs in the browser.

Works best on modern browsers (Chrome, Edge, Firefox).
